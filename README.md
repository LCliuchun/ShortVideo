# ShortVideo

一个基于七牛云开发的短视频app,有美颜效果,支持多种滤镜效果,支持切换背景音乐,支持多视频拼接。
两种方式实现短视频滑动播放

        1.每个item都放videoView 监听recyclerView 的 addOnScrollListener

        2.recyclerView 加一个videoView  ,recyclerView滑动的时候,让videoView切换视频源播放。
